# 📝 To-Do List CLI App (Python)

A simple and efficient **command-line To-Do List application** built using Python.
This project demonstrates real-world programming concepts such as file handling, modular design, and user interaction.

---

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ❌ Delete tasks by number
* 💾 Save tasks to a JSON file
* 🔄 Automatically load saved tasks on startup
* ⚠️ Input validation and error handling
* 🧩 Clean modular structure using functions

---

## 🛠️ Technologies Used

* Python 3
* Built-in modules:

  * `json` (for data storage)
  * `os` (for file handling)

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/dinushan02/todo-list-cli-python.git
```

2. Navigate to the project folder:

```bash
cd todo-list-cli-python
```

3. Run the application:

```bash
python main.py
```
